To host this yourself for free:

### 1. Create a Repository
- Create a **Public** repository on GitHub named `portfolio`.
- **Note**: GitHub Pages is free only for **Public** repositories.

### 2. Upload Files
You have two easy ways to upload your files:

**Method A: Drag & Drop (Easiest)**
1. Open your repository on GitHub in your browser.
2. Click the **"Add file"** button and select **"Upload files"**.
3. Open your local project folder: `C:\Users\ADMIN\.gemini\antigravity\scratch\portfolio-resume`
4. Drag all files (`index.html`, `style.css`, `main.js`, `hero-bg.png`, `README.md`) into the GitHub upload area.
5. Click **"Commit changes"**.

**Method B: GitHub Desktop or Git CLI**
If you have Git installed, you can use the terminal:
```bash
git init
git add .
git commit -m "First commit"
git branch -M main
git remote add origin https://github.com/robinjeyapaul345/portfolio.git
git push -u origin main
```

### 3. Enable GitHub Pages
1. In your repository on GitHub, go to **Settings > Pages**.
2. Under **Build and deployment > Branch**, select `main` (and `/root` folder).
3. Click **Save**.
4. Your site will be live at `https://robinjeyapaul345.github.io/portfolio/` within minutes!
